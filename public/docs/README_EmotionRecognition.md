# Emotion Recognition Android App

A real-time emotion detection and classification application for Android, leveraging computer vision and deep learning.

## 🚀 Key Features
- **Real-time Detection**: Captures and processes facial expressions in real-time using the camera.
- **Emotion Classification**: Detects and classifies human emotions such as Happy, Sad, Angry, Neutral, Surprise, etc.
- **On-Device Processing**: Utilizes TensorFlow Lite for efficient, low-latency inference directly on the mobile device.
- **User-Friendly Interface**: Built with modern Android UI components for a seamless experience.

## 🛠️ Technology Stack
- **Languages**: Java, Kotlin
- **Platform**: Android SDK
- **Machine Learning**: TensorFlow Lite
- **Computer Vision**: OpenCV / CameraX (for frame capture and processing)

## 📋 Project Status
This was developed as a specialized mobile project to demonstrate the integration of AI models into Android ecosystems.
